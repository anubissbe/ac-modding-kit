# Support

This is an unofficial volunteer project. Support is best-effort and does not replace
support from the game developer, Steam, or a mod author.

## Where to ask

- Usage and design questions: [GitHub Discussions](https://github.com/anubissbe/ac-modding-kit/discussions)
- Reproducible toolkit defects: use the GitHub bug report form
- Ancient Cities community knowledge: [Steam Workshop discussions](https://steamcommunity.com/workshop/discussions/18446744073709551615/?appid=667610)
- Confidential security problems: follow [SECURITY.md](SECURITY.md)

Before asking, search existing issues and discussions and test with the newest toolkit
revision. Include the toolkit version, Ancient Cities build, operating system, test
date, exact command, sanitized output, and minimal reproduction steps.

## Protect your data

Back up valued saves outside the game directory and reproduce problems on a new,
disposable save. Mods can corrupt or permanently alter saves and can affect or disable
achievements. Redact usernames, local paths, Steam identifiers, tokens, and personal
information from logs.

Do not upload game binaries, extracted game assets, other authors' Workshop content,
or files derived from binary reverse engineering. Reports containing material that
cannot legally or safely be reviewed may be closed and attachments removed.

## What maintainers cannot promise

Static checks cannot guarantee that a mod is safe, compatible with the current game
build, or accepted by Steam Workshop. Maintainers cannot recover saves, restore
achievements, moderate Steam, or provide support for pirated or modified executables.
